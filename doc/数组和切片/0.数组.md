1. 数组是T存储在连续内存中的`相同类型`的对象的集合. 数组是使用方括号创建的[]

   >`在rust 中，数组不仅包括类型还有数组的长度`

   >`与其他语言不一样的是，数组长度在编译时就也确定，无法修改数组中的长度`

   ```rust
   // 数组不仅包括类型，还有长度
   let array = [0u8; 64];//0u8是无符号整数类型的简写，长度为 8 位，值为 0。0是值，u8是类型。


   let slice: &[u8] = &array;
   ```
