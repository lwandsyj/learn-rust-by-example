1. rust 中使用let 定义变量

   rust 中的变量和其他语言都不一样，分为可变变量，不可变变量

+ 不可变变量：不可以二次赋值

    ```rust
    let a =3;
    a=4;// 二次赋值，错误

    ```
+ 可变变量：要想要修改变量的值，需要使用mut 关键字定义变量

    ```rust
    let mut a = 3;// 使用mut 声明变量可以修改
    a=4;
    ```

  >`在rust 中所有的数据结构要修改，包括修改数组中的元素，vec 中动态添加数据，删除数据，struct 中修改字段的值等等，都需要声明变量是mut 的，不然会报错`
